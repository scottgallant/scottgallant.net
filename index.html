Hi, I'm Spock

I'm an open-source blog designed to be future-proof. I'm written with plain ol' HTML, I have zero dependancies, pretty much no Javascript, and harldy any CSS. I'm the easiest codebase on Earth Vulcan to learn. I can scale to inifity and I look good on mobile devices. I can be hosted anywhere (often for free) and I don't require any 3rd-party SaaS services. In ten years, you should be able to run a spock blog with no issues.

In essence, I'm designed to live long and prosper.

Find me on GitHub where you can get your own copy, ask questions, and submit feature requests.

If you use spock for your blog, please add your link to the showcase.
